Coding Education

Jelajahi dasar-dasar coding dan bahasa pemrograman yang esensial dalam dunia cyber security. Memahami kode adalah kunci untuk mengidentifikasi kerentanan dan membuat pertahanan yang kuat.

Bahasa Pemrograman Penting

  • Python: Sangat populer untuk scripting, otomatisasi, dan pengembangan alat keamanan (e.g., *network scanning, web scraping).
  • C/C++: Penting untuk memahami cara kerja sistem operasi dan aplikasi tingkat rendah, serta menganalisis malware dan reverse engineering.
  • JavaScript/HTML/CSS: Fundamental untuk web application security, memahami kerentanan XSS (Cross-Site Scripting) dan injeksi.
  • SQL: Penting untuk menguji keamanan database dan memahami serangan SQL Injection.

Konsep Dasar

Fokus pada struktur data, algoritma, dan pemrograman berorientasi objek (OOP) untuk membangun logika yang efisien dan aman.

Edukasi Malware

Malware (Malicious Software) adalah ancaman utama. Memahami jenis-jenisnya adalah langkah pertama untuk melindungi diri dan sistem.

Jenis-Jenis Malware

  • Virus: Menempel pada file program dan menyebar saat file dieksekusi.
  • Trojan: Menyamar sebagai perangkat lunak legal untuk mencuri data atau memberikan akses belakang.
  • Ransomware: mengenkripsi file pengguna dan meminta tebusan untuk mendekripsinya.
  • **Spyware:** Diam-diam mengumpulkan informasi pengguna (kata sandi, aktivitas browsing).

Pencegahan Dasar

Selalu gunakan perangkat lunak antivirus/antimalware, hindari mengunduh dari sumber yang tidak terpercaya, dan jangan membuka lampiran email yang mencurigakan.

Prinsip Cyber Security

Keamanan siber adalah praktik melindungi sistem, jaringan, dan program dari serangan digital. Ini didasarkan pada konsep **CIA Triad**.

CIA Triad (Konsep Kunci)

  • **Confidentiality (Kerahasiaan):** Memastikan data hanya dapat diakses oleh pihak yang berwenang. (*Contoh: Enkripsi, otorisasi.*)
  • Integrity (Integritas): Memastikan data akurat, lengkap, dan tidak diubah tanpa izin. (Contoh: Hashing, tanda tangan digital.)
  • **Availability (Ketersediaan): Memastikan pengguna yang sah memiliki akses ke sistem dan data saat dibutuhkan, (Contoh: Backup, perencanaan pemulihan bencana.)

Ancaman Umum

phishing, serangan DDoS (Distributed Denial of Service), dan serangan *Man-in-the-Middle, adalah beberapa ancaman yang harus diwaspadai dalam pengamanan sistem.
